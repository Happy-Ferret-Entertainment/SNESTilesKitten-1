

SNESTilesKitten is a tool to view/extract/inject graphic tiles in a SNES Rom file. 
It allows you to use SNES address (support HiROM/LoROM) so you can use it to discover
tile location directly from ASM trace.


Find more information https://skarsnik.github.io/SNESTilesKitten/
